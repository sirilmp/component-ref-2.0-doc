# Documentation Website

This is the source code for the premium documentation website of `vpcr`.

## Development

```bash
cd docs-site
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features
- **Premium Design**: Built with React and Vanilla CSS.
- **Animations**: Uses Framer Motion for smooth transitions.
- **Comparison**: Includes a "Why this?" section focused on the advantages over Lovable Tagger.
